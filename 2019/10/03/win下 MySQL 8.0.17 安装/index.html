win下 MySQL 8.0.17 安装

1、在官网下载MySQL

地址如下 https://dev.mysql.com/downloads/mysql/

我下载的是64位系统的zip包。下载后解压地址：D:\Mysql

2、配置环境变量

  • 添加系统变量

在其他博客里看到要在path里添加%MYSQL_HOME%\bin;，不过我没有添加也成功安装，暂时没有出现什么问题。

3、新建配置文件

在D:\Mysql\mysql-8.0.17-winx64目录下新建my.ini配置文件，并将以下内容复制粘贴进去。
注意里面有两个需要修改的地方：mysql安装目录和mysql数据库的数据存放目录。

其中，mysql安装目录就是刚才zip包解压的目录，也就是MySQL的安装路径，在这里是D:\Mysql\mysql-8.0.17-winx64，数据库的数据存放目录指的是MySQL的数据库文件放在什么路径下，这里的目录会在接下来的配置中自动生成，如这里的D:\MySQL\Database，接下来会在D:\MySQL下自动生成Database目录。

若在配置之前Database目录就已经存在，可能会报Warning的错，将目录删掉或修改my.ini文件中的目录，重新配置一下就好了。

4、自动生成MySQL初始化配置文件

以管理员身份打开cmd，在MySQL安装目录的bin目录下执行命令：

mysqld --initialize --console
初始化配置文件

记得截图保存下打印下来的东西：

画横线的就是登陆时的密码，接下来登陆数据库时需要用到

如果不小心关掉了= = ，只需要删掉data目录，然后重新执行一遍初始化命令，又会重新生成

5、安装服务

在MySQL安装目录的bin目录下执行命令：

mysqld --install [服务名]
服务名指的是给服务命名，可以不写，默认为mysql。

如果电脑上需要安装多个MySQL服务，就可以用不同的名字区分，如mysql1,mysql2

6、启动、停止服务与登录

net start mysql 启动MySQL服务
net stop mysql 停止服务
mysql -u root -p 登录：这时候会提示输入密码，输入上面初始化时生成的密码，即可成功登录

若提示「发生系统错误5 拒绝访问」 fuhao，则用管理员身份打开cmd，重新输入命令即可。

7、修改密码

在登录后执行命令：

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '新密码'
